# Welcome to Your Deployery Instance

## Package management

- System packages: `sudo apt install <package>`.
- Node packages: `npm install <package>`.
- Python packages: `pip install <package>`.

## Keyboard shortcuts

Browsers do not allow sites to intercept some tab-management shortcuts such as `Ctrl+W` and `Ctrl+T`.

For the closest native IDE experience in the browser, install the page as an app.
